# 运行
Linux Ubuntu
```bash
$ git clone https://github.com/vintingb/daomengKJ.git
$ sudo pip3 install -r ./requirements.txt
$ python3 ./daomeng.py
```
Windows
```bash
直接releases下下载运行
```
